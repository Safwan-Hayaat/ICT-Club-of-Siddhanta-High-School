Fields of ICT

ICT has an extreme variety of different uses and forms. Ranging from entertainment, education, work, medicine and just life in general. So, here are a select few fields of ICT to get a basic understanding of them.


Coding

Coding in its most basic form means writing many specific instructions to a computer so that the computer can execute those instructions.

Coding is one of the most important and influential parts of ICT, it has changed and shaped the world as we know it. Below there are the 3 programming languages used to make a website.

HTML

See the Pen Rock Paper Scissors HTML by Safwan Hayaat (@Safwan-Hayaat) on CodePen.

HTML (HyperText Markup Language) is a markup language developed and released by Tim Berners-Lee and scientists at CERN. It was released in 1993. It is known as the “Structure of the web”. Because with plain HTML, we can make websites that have text, buttons, images and more. But they won’t look good, nor will they function.

For example, There is a rock paper scissors game at the top. Although we can’t play it and there is no design. The text and the buttons are still there. This is why HTML is still very important. Because without HTML we wouldn’t be able to display anything on a website.

HTML is used to add text, buttons, links, images, videos and more to a website.

Due to the limitations of HTML, there are 2 other coding languages that are used alongside it to make websites.

CSS

See the Pen Rock Paper Scissors CSS by Safwan Hayaat (@Safwan-Hayaat) on CodePen.

CSS (Cascading Style Sheet) is a style sheet language developed and released by Håkon Wium Lie and scientists at CERN. Which was released in 1996. It is used to give designs to a website. Because of this, CSS is the main coding language for making designs on the web, and it is used along with HTML to give a website life and personality. But CSS has its own limits. CSS is a language that can only make designs to existing code. which means that using CSS, one cannot add text, images, or any functionality using CSS. Which is why it is used alongside HTML to make a website.

At the top we can see that the same rock paper scissors game we saw before now has animations and colors. Although the game still doesn’t work, it looks much better now than it used to look.

CSS is used to add animations, colors, custom shapes, fonts, text styles and more to a website.

JS

See the Pen Rock Paper Scissors JS by Safwan Hayaat (@Safwan-Hayaat) on CodePen.

JS (JavaScript) is a programming language developed by Brenden Eich and released by Netscape in 1995. JS is used to add interactivity and functionality to a website. JS was made to be used with HTML and CSS, as it cannot add designs to a website, and it lacks many things that can be done with HTML in just 1 line. But JS is still very useful to make things function and work on a website.

At the top we can see that the rock paper scissors game looks the same, but it now works. By pressing a button, we can select our move, the computer selects the next move on its own, and then we get the result.

JavaScript is used to add functionality, run scripts, do math and calculations, run algorithms, change a website’s interface in realtime and more.

This was only a basic breakdown of 3 coding languages out of the thousands of other coding languages that exist out there. What we can learn from this is that every coding language serves its own purpose, some may be used to build apps, others for making games and others to make websites. Other famous programming languages include Java (different from JavaScript), Python, C, C++ and many more.

Word Processing

A Word Processor is a kind of Software or Hardware made to edit, save, format or design a document. Word Processing is a very important aspect of ICT which is used very frequently in office spaces for writing documents. Below there are the 2 most popular word processors.

Microsoft Word

Microsoft Word is a Word Processor developed and released by Microsoft. It was released in 1985 for Xenix systems. Microsoft Word is the most famous Word Processor in the world. It is an offline Word Processor (meaning you have to download it to use it) that is a paid-service. Despite this, Microsoft Word is the industry standard, simply because it is easy to use and has been out for over 40 years. When it first came out, it was a game-changer for the industry, it had basic abilities like changing text styles and size (which was a feature almost every Word Processor had at the time) it also had more advanced features for the time, like spellcheck, which checked if words were spelled correctly or not. But it was most praised for how easy it was to learn. Word was a WYSIWYG (What You See Is What You Get) software, which means every part of the editing, formatting etc. was visual. Since then Microsoft has updated Word hundreds of times, adding features like adding graph, symbols, changing page layout, many design choices, being able to check the word count of a document, font changing and many more.

Google Docs

Google Docs is a Word Processor released by Google in 2006. It is the most famous online Word Processor and 2nd most famous Word Processor behind Google. Google Docs is a free and online software which makes it convenient for students, new authors and anyone else who aren’t able to afford other paid-services. Google Docs also has its own unique features. Because it is an online software, it has the ability to let multiple users collaborate and every user can see each others edits in real-time. Google Docs also has detailed pre-made professional templates for almost every type of document. It also has most features that Microsoft Word has. As for which word processor is better it is highly dependent on situation and field. It is up to you, the reader to decide which is better to use. Because depending on your needs and the type of document you are making, either of these software can be better.

This was a basic look into only 2 word processors out of many.

Word Processors are used by teachers, researchers and students to make papers and documents easily. It is used by office workers and businessmen to make notes and meeting plans. They are used by authors, writers and journalists to write or make books. Just to name a few examples.

Other famous word processors include LibreOffice Writer, OpenOffice Writer and more.
